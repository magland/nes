# nes

An adventure game

## Installation

During development, do the following

### Clone this repo

```
git clone https://github.com/magland/nes
```

### Install in development mode

```
cd nes
pip install -e .
```

### For subsequent updates

```
cd nes
git pull
pip install -e .
```

## Running

Once installed, just open a terminal and run:

```
nes
```
